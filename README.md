## Project Overview

Dr. Echo is an AI-assisted reporting system designed to support radiology workflows by improving reporting efficiency, consistency, and accuracy. The system integrates artificial intelligence with a user-friendly interface to assist clinicians in generating structured radiology reports.

This project was developed as a **Final Year Project (FYP)** in a **group setting**, with clear role distribution among team members.


## Problem Statement

Radiology reporting can be time-consuming and prone to inconsistencies due to manual processes and high workloads. There is a need for a system that can assist radiologists by streamlining report generation while maintaining clarity and accuracy.


## Solution

Dr. Echo provides:

* AI-assisted support for radiology reporting
* A structured and intuitive web-based interface
* Improved workflow efficiency and report consistency


## Technologies Used

**Frontend:**

* Languages: HTML, CSS, JavaScript
* Hosting: Firebase Hosting
* Design & Prototyping: Figma, MockFlow

**Backend:**

* Framework: Python FastAPI
* Server: Uvicorn (ASGI)
* AI Model: Google Gemini (Large Language Model)
* Data Validation: Pydantic

**Backend-as-a-Service (BaaS):**

* Database: Cloud Firestore (NoSQL Document Database)
* Authentication: Firebase Authentication
* File Storage: Firebase Storage
